# BlinkIt Clone - E-commerce Web Application

## Description

This is a BlinkIt-style eCommerce web application built with the MERN stack (MongoDB, Express, React, Node.js).  
It allows users to browse products, add them to their cart, and manage their accounts. There are two types of users: **Admin** and **Customer**.  
Product images are uploaded and managed via **Cloudinary**, providing a fast and scalable media storage solution.

---

## Features

### General Features

- Responsive and user-friendly interface
- Role-based authentication: Admin & Customer
- User login/signup via email and password
- Reset password
- Product listing with images, descriptions, and prices
- Search products by name
- Filter products by category or price range
- Images uploaded and stored using **Cloudinary**

### Admin Features

- Add, edit, or delete products (images uploaded via Cloudinary)
- View all products in the catalog
- Manage categories
- Access analytics and dashboard (if implemented)

### Customer Features

- Browse and search products
- Add/remove items to/from cart
- View cart and total price
- Place orders
- Cancel orders
- View order history

---

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Image Storage:** Cloudinary
- **Authentication:** JWT-based email/password login
- **Deployment:** Vercel for frontend; backend connected via API
